Testifj

Testing for Java

What is this?

Testifj is a testing framework for Java 8 that is typically used from JUnit/TestNG to write expressive tests.

How do I get it?

Testifj is not feature complete yet and is there for not yet available in mvnrepository.com. As soon as a feature complete (isch) version is available it'll be deployed as an alpha-version.

How do I use it?

Basic test cases

import static org.testifj.Expect.*;
public class MyTestCase {
    @Test
    public void theSimplestPossibleUse() {
       expect(true).toBe(false);
    }
}
